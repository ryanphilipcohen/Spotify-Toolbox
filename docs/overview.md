# Spotify Toolbox

## Overview
Spotify Toolbox is an enhanced playlist creation tool that works almost fully offline with your Spotify library. It extends Spotify's native playlist functionality by adding powerful features like tags and templates, giving you more control over how your music is organized and played.

## Key Features

### Offline Creation
- Fetch and store your Spotify liked songs locally
- Create and modify playlists without an internet connection
- Sync your changes back to Spotify when ready

### Smart Tagging System
- Add custom tags to any song in your library
- Filter and organize music using multiple tags
- Use tags to create dynamic playlist templates

### Dynamic Templates
Templates are the core of Spotify Toolbox's playlist creation system. Unlike traditional playlists, templates can:
- Include tag-based song selection
- Generate multiple unique playlists from a single template
- Save templates for repeated use

## How It Works

### Creating a Workout Playlist Example
Let's say you want to create the perfect workout playlist that maintains a consistent energy level:

1. Tag your songs by intensity:
   - Low intensity: "Cool down" songs
   - Medium intensity: "Steady state" songs
   - High intensity: "Peak performance" songs

2. Create a template with an energy-managing pattern:
   - Define a rotation of low → medium → high intensity songs
   - Add multiple songs for each intensity level

3. Generate unique playlists:
   - Each new instance of your template creates a fresh playlist
   - Songs are automatically arranged to maintain your desired intensity pattern
   - Every workout gets a new mix while maintaining the perfect energy flow

This solves the common problem of random shuffle either clustering high-intensity songs or creating unpredictable energy patterns in your workout.
